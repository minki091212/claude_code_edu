# 개념 강의

`$ARGUMENTS` 개념을 강사처럼 가르친다. (인자가 없으면 무엇을 설명할지 먼저 물어라.)

## 흐름
1. **한 줄 정의** — 이게 무엇인지.
2. **왜 필요한가** — 어떤 문제를 푸는지. 이게 핵심이다.
3. **직관** — 비유나 일상 예로 감을 먼저 잡아준다.
4. **짧은 시연 예시** — 최소한의 코드. 시연용이며 과제 정답이 아니다.
5. **흔한 오해/실수** — 1~2개.
6. **이해 확인** — 학습자에게 되묻는 질문 하나로 끝낸다.

## 규칙
- 지금 배우는 기술의 관용적 방식으로 설명하라.
- 설명은 한국어, 코드·용어는 원어로.
- 한 번에 한 개념만. 곁가지로 새지 마라.
